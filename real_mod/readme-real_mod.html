Installation

Windows
Real Mod is packaged and installed with WeiDU and is distributed as a self-extracting archive. To install, simply double-click the archive and follow the instructions on screen.

Alternatively, the files can be extracted from the archive using WinRAR or 7zip and moved to your game folder. Your game folder should then contain a real_mod folder and setup-real_mod.exe. To install, double-click setup-real_mod.exe and follow the instructions on screen.

Please run setup-real_mod.exe in your game folder to reinstall, uninstall, or otherwise change components.

Mac OS X
Real Mod is packaged and installed with WeiDU and is distributed as a compressed archive. If the mod was previously installed, uninstall it before extracting the new version. Extract the contents of the archive and move them to your game folder. Your game folder should then contain a real_mod folder, setup-real_mod, and setup-real_mod.command. To install, simply double-click setup-real_mod.command and follow the instructions on screen.

Linux
Extract the contents of the mod archive to your game installation directory.

Download the latest version of WeiDU for Linux from weidu.org and extract (at least) the WeiDU, weinstall and tolower executables to a $PATH directory (e.g. /usr/bin).

Following that, open a terminal, cd to your game installation directory, run 'tolower' and answer Y to both queries. You can avoid running the second option (linux.ini) if you've ran it previously in this directory. To save time, the contents of this mod archive are already tolowered, so you can avoid running the first option as well (lowercasing file names) if this is the only mod you've extracted since the last time you lowercased file names. If you're unsure, running tolower and choosing both options is the safe bet.

After that, run weinstall real_mod from your game installation directory to install or uninstall the mod.
